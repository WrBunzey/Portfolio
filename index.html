Tulip

For my Software Engineering capstone project, I worked with a classmate to create a simple messaging application named TwoLip. This application used Amazon Web Services for the backend to handle data storage and networking capabilities. We used services such as Amazon SQS for message queuing, DynamoDB for message storage, SNS for notifications, and S3 for image/file storage. TwoLip supports messaging one or more people, profile customization, and file/image embedding in messages. The frontend for Twolip utilizes the Electron API to create a desktop application using HTML, CSS, and JavaScript, allowing for the creation of a customizable GUI using webdev tools and supporting both desktop and web browser apps. For the branding of this application, we decided on naming it TwoLip, referring to tulips and mouths, representing both growth and communication. Our logo reflects this naming convention.

Vigenere Cipher Password Manager

I created this program intending to utilize command line arguments and cryptography for encryption while I was learning Python at the HACTC. My program can read and write encrypted passwords to a CSV file stored in the same directory. When you enter a password, it will be encrypted with a Vigenere cipher using the key you provide and saved with a provided title. A Vigenere cipher encrypts each letter of a phrase using a separate Caesar cipher corresponding to the value of a letter in the encryption key.
